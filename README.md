### ExamAssistant

Идея проекта: помощник по подготовке к экзаменам в вузе.

Пока MVP:



Сценарии использования:

1. Пользователь загружает pdf-файлы конспектов, теха лекций и всего такого.

2. Пользователь пишет какой-то запрос и ИИ может использовать загруженные файлы для этого.



Архитектура:

Для апи пока будут два эндпоинта:

1. Для загрузки документа.

2. Для запроса.



1. Загрузка документа:

- Получаем документ

- Разбиваем его на чанки, сохраняя мета-информацию (номер страницы, название файла и тд). Разбиваем по-умному, не просто по n элементам, а по предложениям или абзацам, но не слишком большими.

- Эти чанки с мета-информацией превращаем в эмбеддинги и закидываем в векторную базу данных.



2. Запрос:

- Получаем запрос от пользователя

- Отправляем его в нашу гпт

- если гпт не нужен дополнительный контекст для ответа (из загруженных файлов), то просто отвечаем и возвращаем этот ответ.

- если нужен контекст. Просим гпт написать предложения для поиска в векторной бд, и превращаем его в эмбеддинг. Дальше ищем n ближайший векторов к этому предложению. Дальше заново строим запрос в гпт, берем запрос пользователя и добавляем найденные данные. Дальше возвращаем ответ от гпт пользователю.



Планирую использовать следующий стек технологий:

1. Для апи - FastAPI

2. Для векторной базы данных - Qdrant

3. Для работы с ИИ - LangChain или LangGraph.
